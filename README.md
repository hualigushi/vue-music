# vue-music

> 仿QQ音乐APP，数据使用QQ音乐的接口爬取
> 技术栈 ：vue+vuex+vue router+axios

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# server port at localhost:8080
node prod.server.js
